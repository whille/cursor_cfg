# 定植时的蔬菜秧苗苗龄

（前期）**产量降低**造成的。**苗龄较大**，**前期产量较低**，但**中后期产量高**，**生育期相对地延长**。所以日光温室冬春茬果菜类栽培，**苗龄不宜太大**，而温室大棚春茬果菜类栽培以尽量争取提早采收期，需要培育较大的秧苗。
