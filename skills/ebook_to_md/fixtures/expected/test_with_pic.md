# 长后坡半拱形日光温室

多为 **5.5～6 米**，**中脊高 2.3～2.4 米**，**后坡长 3 米**，**后屋面厚度 60～70 厘米**。这种温室的**优点是造价低、保温性能好**，**缺点是采光面积小**，长后坡下面**光照弱**，特别在春秋两季，**后坡下弱光带较宽**。近几年不断进行改进，将**中脊高度提高到 2.7 米**，**后坡投影长度缩到 2～2.2 米**，即兼顾了保温效果，又增加了采光量，进一步改善了温光性能（图 5）。这种温室在北纬 **41°以北地区**冬季可生产叶菜，在 **40°以南地区**可以生产喜温的**果菜类蔬菜**。

## （2）短后坡高后墙半圆拱型日光温室

这种温室是在总结长后坡矮后墙温室优缺点基础上改进的，由于**加长了前屋面、缩短了后坡、提高了中脊高度**，**采光面加大，透光率显著提高**，后坡下面**光照强度增加 20% 左右**。在光照好的地区，中午前后室温比长后坡温室高，次日揭苫时**室内最低温度几乎与长后坡温室一致**。**冬季也能进行果菜类生产**。近年来，这种温室在**辽宁中北部、吉林、河北和内蒙古**等地发展很快。
